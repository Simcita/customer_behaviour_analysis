# Customer Shopping Behaviour Analysis

## Overview

This project analyzes customer shopping behavior using **Python, SQL, and Power BI**. The goal is to explore purchasing patterns, clean and transform the dataset, run analytical queries, and visualize insights through an interactive dashboard.

The workflow demonstrates a **complete data analysis pipeline**, from raw data exploration to business-ready reporting.

---

## Dataset

The dataset used in this project contains customer shopping behavior information such as:

* Customer demographics
* Purchase frequency
* Product categories
* Transaction values
* Shopping patterns

The dataset is loaded and analyzed using **Python (Pandas)** and further queried using **PostgreSQL** for deeper insights.

---

## Tools & Technologies

This project uses the following tools:

* **Python** – Data loading, cleaning, and exploratory data analysis
* **Pandas & NumPy** – Data manipulation and statistical summaries
* **PostgreSQL** – Querying structured data using SQL
* **Power BI** – Building interactive dashboards and visualizations
* **Gamma** – Creating presentation slides
* **Jupyter Notebook** – Running and documenting the analysis

---

## Project Steps

### 1. Data Loading

The dataset is loaded into a Jupyter Notebook using **Pandas**.

Tasks performed:

* Import dataset
* Inspect structure
* Review initial records

---

### 2. Exploratory Data Analysis (EDA)

EDA was conducted to understand the dataset and identify patterns.

Key activities:

* Summary statistics
* Data type inspection
* Missing value detection
* Distribution analysis
* Customer purchasing patterns

---

### 3. Data Cleaning

Data preprocessing was performed to ensure quality and consistency.

Cleaning steps included:

* Handling missing values
* Fixing inconsistent formats
* Removing duplicates
* Preparing data for SQL analysis

---

### 4. SQL Analysis (PostgreSQL)

The cleaned dataset was imported into a **PostgreSQL database**.

SQL queries were used to:

* Identify top purchasing categories
* Analyze spending trends
* Segment customers
* Generate aggregated insights

---

### 5. Power BI Dashboard

A **Power BI dashboard** was built to visually present insights from the analysis.

Dashboard features include:

* Sales distribution
* Customer purchase trends
* Category performance
* Key metrics and KPIs

The dashboard enables quick and intuitive exploration of the results.

## How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/customer-shopping-behaviour-analysis.git
cd customer-shopping-behaviour-analysis
```

---

### 2. Install Dependencies

```bash
pip install pandas numpy psycopg2 jupyter
```

---

### 3. Run the Notebook

Start Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```
Customer_Shopping_Behaviour_Analysis.ipynb
```

---

### 4. Run SQL Queries

1. Import the dataset into **PostgreSQL**
2. Execute the SQL queries included in the project
3. Use the results for visualization in Power BI
